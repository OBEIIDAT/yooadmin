# YOOAdmin

YOOAdmin is a modern WordPress admin interface enhancement focused on clarity, organization, and usability.

## Features

- Cleaner admin layout
- Improved notification handling
- Distraction-free interface refinements
- Lightweight and performance-friendly
- Works without requiring a license

## Installation

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the WordPress admin panel.
3. Configure settings under YOOAdmin.

## Requirements

- WordPress 6.0+
- PHP 7.4+
